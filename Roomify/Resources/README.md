**May 22nd idea talk**
- First initial talks & idea setting
- Viability unknown just yet

*Idea*
- Using Threejs, create a room customizing application. 
- Low poly art style

*Implementations*
- Tile based room. (Constant size. 8x8)
- Room and furniture blocks
- Position must be normalized to fit in tiles.
- Pan and tilt (Navigate) with mouse drag
- Usable furniture blocks are listed on the right side of screen
- Can drag and place them on room
- Simple collision logic. No other furnitures on the same tile
- Correct lighting

*Additional Goals*
- Non-constant room size. User can change them.
- User can change the color of wall, floor, and furnitures.
